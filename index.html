    Our Journey

    Quirino High School participated in the iamtheCODE pilot program, a global initiative dedicated to empowering young women through coding, digital literacy, and STEAMD education. As one of the pilot schools, Quirino High School provided its students with the opportunity to gain hands-on experience in coding, technology, and collaborative problem-solving over a 12-week program.

    Timeline

    Learnings

    CHANNEL

    1. Welecome to the Program

    This module introduces the overall course, its objectives, and what learners can expect to gain by the end of the program. It helps students understand the importance of digital skills and prepares them for the lessons ahead by setting clear goals and expectations.

    ASPIRE JOURNEY

    2. Learn to Code in HTML

    This module focuses on introducing students to HTML as the basic structure of a website. Learners are taught how to create and organize web content using elements such as headings, paragraphs, images, and links. Through simple activities and practice tasks, students begin building their own web pages and understand how content is arranged and displayed on the internet.

    ASPIRE JOURNEY

    3. Learn to Code in CSS!

    In this module, students explore how to design and improve the appearance of their web pages using CSS. They learn how to apply colors, fonts, spacing, and layouts to make websites more visually appealing and user-friendly. The lessons help learners understand the importance of design in communication and allow them to customize their pages to reflect creativity and clarity.

    ASPIRE JOURNEY

    4. Learn to Code in Javascript!

    This module introduces learners to JavaScript as a tool for making websites interactive and dynamic. Students practice using simple scripts to control buttons, display messages, and respond to user actions. By learning basic programming logic and functions, they begin to understand how websites can move beyond static content and provide engaging experiences for users.

    ASPIRE JOURNEY

    5. Learn to Code in HTML, CSS, amd Java Skills!

    This module focuses on building websites using HTML for structure, CSS for design, and JavaScript for interaction. Students learn how web pages are created and how these three technologies work together to create functional and visually appealing websites.

    ASPIRE JOURNEY

    6. Learn to Code in Python!

    In this module, students are introduced to Python programming. It covers basic concepts such as variables, functions, and strings, and uses tools like Jupyter Notebooks to help students practice coding in an interactive and beginner-friendly environment.

    LINK

    7. Getting to know the Wellbeing Code

    This module introduces the Wellbeing Code and its purpose of providing a safe and supportive space for young people. Students learn about the importance of wellbeing, open discussion, and community awareness in daily life.

    LINK

    8. Sustainable Development - SDG's

    Module 8 focuses on the United Nations‚Äô Sustainable Development Goals. It helps students understand global issues such as poverty, inequality, climate change, and peace, and shows how individuals and technology can contribute to a more sustainable future.

    CHANNEL

    9. What is Design Thinking?

    This module introduces design thinking as a problem-solving approach centered on understanding people‚Äôs needs. Students learn how empathy, creativity, and innovation can be used to develop better products, services, and solutions.

    CHANNEL

    10. Commmunication Skills

    In this module, students learn how to communicate more effectively through listening, speaking, writing, and body language. The lessons emphasize the importance of clear and confident communication in personal, academic, and professional settings.

    CHANNEL

    11. Presentation Skills

    Module 11 focuses on developing clear and engaging presentations. Students learn techniques for organizing ideas, delivering messages confidently, and making presentations more effective and memorable.

    CHANNEL

    The final module introduces basic project management skills. Students learn how to plan tasks, manage time, and work within a set scope and budget, helping them understand how projects are successfully completed from start to finish.

    FAQs
    What is iamtheCODE?
    iamtheCODE is a global movement that empowers youth‚Äîespecially girls‚Äîto learn coding, digital literacy, and STEM skills to achieve the UN Sustainable Development Goals.
    How did Quirino High School implement the program?
    Through 12 weeks of hands-on coding workshops, mentorship, and collaborative activities that developed both technical and soft skills among students.
    What skills did students gain?
    Students learned HTML, CSS, and JavaScript, as well as teamwork, leadership, and communication skills vital to success in technology and beyond.
    How does the program promote education and equality?
    By ensuring equal access to digital learning opportunities for both male and female students, fostering inclusion and empowerment through technology.
    How can other schools join or replicate the initiative?
    Schools can reach out to the iamtheCODE foundation or collaborate with educational organizations that promote coding and digital inclusion.
    How did coding help students beyond technology?
    Coding improved students' patience, focus, logical thinking, teamwork, and confidence. It taught them how to solve problems, express creativity, and work collaboratively, skills that are useful in school, careers, and daily life.
    How can I start my coding journey?
    You can start by exploring free online resources like W3Schools, Khan Academy, and Code.org. Even practicing simple HTML or Python can begin your journey. Joining school ICT clubs or participating in programs like iamtheCODE also helps.
    What is the relevance of this program?
    The program is relevant because it prepares students for a world driven by technology. It strengthens digital literacy, promotes equality, and encourages students to become future innovators who contribute positively to society.
    How does our project support the Sustainable Development Goals?
    Our project supports SDG #13: Climate Action by raising awareness about environmental issues and encouraging sustainable practices.
    What did we learn from this experience?
    We learned how to code, collaborate, communicate, and think creatively. We also gained a deeper understanding of real-world issues and the importance of contributing to the SDGs. Most importantly, we learned that technology can be a powerful tool for change.
